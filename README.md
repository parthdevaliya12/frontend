# 🚀 Parth Devaliya Portfolio

A modern, fully responsive developer portfolio website built using React.js and Tailwind CSS, showcasing my projects, skills, and experience.

---

## 🔗 Live Demo

* https://your-domain.com

---

## ✨ Features

* Modern UI with smooth animations (Framer Motion)
* Dark / Light theme toggle
* Fully responsive (mobile, tablet, desktop)
* Fast performance with lazy loading
* Interactive Hero section with typing animation
* Projects showcase with tech stack & icons
* Skills section with animated UI
* Contact form (Web3Forms integration)
* Scroll-to-top button
* Download CV option

---

## 🛠 Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Framer Motion

### Libraries & Tools

* React Icons
* React Typed
* Web3Forms (Contact API)
